# Clone x Niche Integration

NF-L3 maps validated joint clones onto spatial units.

## Spatial Units

- Cellbin
- Tile
- Group
- Metaniche

## Outputs

- Clone composition matrices
- Clone richness, entropy, and dominance summaries
- QC-aware coverage summaries

Tiles are the primary spatial summary because they are non-overlapping.
Groups and metaniches are local-context summaries.
